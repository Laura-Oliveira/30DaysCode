## Problem
Read integers, doubles and strings from input and print the result.

## Key Concepts
- Primitive data types
- Input handling
- String concatenation

## Kotlin Concepts Used
- readlnOrNull()
- Nullable types
- String templates

## Complexity
O(1)

## Notes
Kotlin handles nullable values differently than Java. Using `readlnOrNull()` prevents runtime exceptions when input is missing.
